# 📌 Projeto Java com Spring Boot

## 📂 Estrutura do Projeto

📦 src/

├── controller/ # Gerencia requisições HTTP 

├── service/ # Lógica de negócio 

├── domain/ # Entidades do sistema

├── configuration/ # Configurações da aplicação


🚧 A aplicação também possui uma configuração do profile **dev**, para testá-la em um ambiente de desenvolvimento. 👨‍💻

---

## 🏗️ Tecnologias Utilizadas 🧾

- **Java 21** e gerenciador de dependências **Maven**
- **Dependências:**
  - Spring Boot Dev Tools
  - Spring Web
  - Lombok
- **Métodos HTTP:** `GET` e `POST`
- **Ferramentas para desenvolvimento:**
  - **Postman** → Testes de endpoints
  - **Git & GitHub** → Controle de versão
  - **IDE IntelliJ** → Desenvolvimento

---

## 🎯 Objetivo da Aplicação

🖱️ A aplicação tem por objetivo adquirir mais conhecimentos e exercitar algumas habilidades já adquiridas de **Java** e **Spring Boot**. 😀
